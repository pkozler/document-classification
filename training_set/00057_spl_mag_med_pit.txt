 Nova spustila nový web pro  mladé dívky Krásná.cz  --------------------    Ke zpravodajství: ilustrační FOTO ve Fotobance  ID foto: F201010270281701, F200909230097001   --------------------    Praha 14. ledna (ČTK) - Novinky ze světa módy a života celebrit nebo rady, jak řešit problémy ve vztahu či při péči o vzhled, nabízí mladým dívkám nový lifestylový web Krásná.cz. O rozjezdu internetového magazínu televize Nova dnes informovala ČTK mluvčí Novy Michaela Fričová.   "Primární cílovou skupinou webu Krásná.cz jsou mladé dívky od 14 do 17 let," cituje tisková zpráva Pavla Krbce, ředitele sekce internetu TV Nova.   Teenagerky zde naleznou články o tématech, která je podle mínění redakce nejvíce zajímají. Dozvědí se například, jak si ještě více nezhoršovat své akné, jak "nerozfofrovat" kapesné hned první den nebo jaké vůně nejvíce vzrušují kluky. Web je také informuje, na jakých místech se "nevyplatí" provozovat sex. Součástí webu je také sekce věnovaná radám a názorům čtenářek, k aktuálním tématům se zde mohou vyjadřovat i samotné návštěvnice stránek. Ty mají rovněž možnost založit si zde vlastní blog   Stránka krásná.cz rozšířila seznam internetových projektů provozovaných televizí Nova. Mezi ně patří web www.nova.cz, zpravodajské stránky www.tn.cz, magazín pro ženy www.doma.cz či nově spuštěný videoportál www.voyo.cz.   phg jnd
